# ♛ CheckersVerse

> **Механика, которой 5000 лет — в продукте уровня 2026 года.**

CheckersVerse — первая полноценная **competitive платформа для шашек**.  
Шахматы получили Chess.com с 100M+ пользователями. Шашки не получили ничего сравнимого. **Мы первые.**

| Аудитория | Боль | Что даём |

---

## ⚡ Что внутри

### 🤖 AI противник
- Minimax + Alpha-Beta pruning
- 4 уровня: Лёгкий / Средний / Сложный / Эксперт
- AI Coach — разбор ошибок и упущенных комбинаций после партии

### 🌐 Мультиплеер
- Игра по ссылке — скопировал, отправил, играешь. **Без лобби, без регистрации**
- Polling-синхронизация (WebSocket-ready архитектура)
- Таймер на каждого игрока, чат в партии

### 🏆 XP и уровни
- 10 уровней: Bronze → Silver → Gold → Platinum → Diamond → Grandmaster
- XP за победы, двойные взятия, захват дамок, длительность партии
- Level-up уведомление прямо в модальном окне

### 🎖️ Достижения (10+)
- **First Win** — первая победа
- **Double Killer** — двойное взятие в одном ходу
- **Beat Hard AI** — победа над сложным ИИ
- **Streak Master** — 5 побед подряд
- **King Slayer** — взял дамку соперника
- И другие...

### 🔥 Daily Streak
- Играй каждый день — получай XP-бусты и скины
- Сгорает при пропуске дня

### 🎁 Сундуки с наградами
- 4 типа: **Common / Rare / Epic / Legendary**
- Выдаются после побед в зависимости от сложности и перфоманса
- Внутри: XP-бонусы, скины шашек, темы доски, анимации победы
- Анимация открытия с партиклами

### 📊 Глобальный лидерборд
- Рейтинг ELO по всем игрокам
- Фильтр по городам
- Топ по очкам, винрейту, стрику

### 🎨 Кастомизация
- 6 скинов шашек: Classic, Neon, Fire, **Gold** *(Pro)*, **Emerald** *(Pro)*, Galaxy
- Темы доски
- Анимации победы

## 📁 Структура проекта

```
checkmasters/
├── index.php                # Лендинг
├── leaderboard.php          # Глобальный рейтинг
├── config/
│   ├── app.php              # Bootstrap, helpers, getCurrentUser()
│   └── database.php         # PDO singleton
├── auth/
│   ├── login.php
│   ├── register.php
│   └── logout.php
├── game/
│   ├── play.php             # Игровая страница (AI / PvP / Online)
│   └── lobby.php            # Лобби и создание комнат
├── profile/
│   ├── index.php            # Профиль — XP, уровень, достижения, история
│   └── upgrade.php          # Pro-подписка
├── api/
│   ├── create_room.php      # Создание онлайн-комнаты
│   ├── join_game.php        # Подключение к комнате
│   ├── online_move.php      # Отправка хода
│   ├── game_sync.php        # Polling синхронизация
│   ├── save_move.php        # Сохранение ходов (AI/PvP)
│   ├── end_game.php         # Финал — XP, рейтинг, достижения, сундуки
│   ├── open_chest.php       # Открытие сундука
│   ├── surrender.php        # Сдаться
│   ├── chat_send.php        # Отправка сообщения в чат
│   ├── chat_get.php         # Получение сообщений чата
│   └── activate_pro.php     # Активация Pro-подписки
├── assets/
│   ├── css/
│   │   ├── main.css         # Дизайн-система (токены, утилиты)
│   │   ├── auth.css         # Auth-страницы + общие компоненты
│   │   └── game.css         # Игровые компоненты, скины, лидерборд
│   └── js/
│       ├── engine.js        # Шашечный движок + Minimax AI + Flying King
│       ├── board.js         # UI-контроллер доски
│       └── main.js          # Лендинг JS (анимации, parallax)
└── database/
    └── schema.sql           # Схема БД (users, games, moves, achievements...)
```

## 🔧 Установка

```bash
# 1. Создай БД и импортируй схему
mysql -u root -p < database/schema.sql

# 2. Настрой подключение
# Отредактируй config/database.php — хост, имя БД, логин, пароль

# 3. Разверни на Apache/Nginx с PHP 8+
# Корень → папка проекта

# 4. Открой в браузере
http://localhost/checkmasters/
```
